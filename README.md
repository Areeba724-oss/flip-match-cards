# 🃏 Memory Card Flip Game
A fun, interactive, and responsive web-based memory game where players match pairs of emojis. This project showcases smooth 3D flip animations and dynamic state management using Vanilla JavaScript.

## ✨ Features
* **Dual Themes**: Includes support for both **Light** and **Dark** modes to suit user preference.
* **Difficulty Levels**: Offers two gameplay modes: **Easy (6 Pairs)** and **Hard (9 Pairs)**.
* **Dynamic Scoring**: Features a custom scoring logic based on moves, time, and successful matches.
* **Live Stats**: Real-time tracking of **Time**, **Moves**, and **Score** during gameplay.
* **Matched Animations**: Special visual feedback with green highlights and 3D scaling when a pair is correctly matched.

## 🛠️ Tech Stack
* **HTML5**: Semantic structure for the game grid, menus, and layouts.
* **CSS3**: Advanced usage of CSS variables, Flexbox, Grid, and 3D Keyframe animations.
* **JavaScript (ES6)**: Pure Vanilla JS for game logic, timers, and DOM manipulation.

## 🎮 How to Play
1. Select your preferred **Level** and **Theme** from the Start Menu.
2. Click **Start Game** to begin.
3. Click on cards to flip them and find matching pairs of emojis.
4. Match all pairs in the shortest time and fewest moves to achieve a high score.

**Note**: This project was developed as part of my web development practice to master JavaScript logic and CSS animations.
